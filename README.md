# CSS Projects Portfolio

## Overview

This repository contains a collection of responsive UI projects built while learning and mastering core and advanced CSS concepts. The focus is on **layout systems, responsiveness, component design, and clean styling architecture**, rather than using frameworks.

---

## Projects Included

### 1. Landing Page

**Features:**

* Responsive navbar
* Hero section with call-to-action
* Features grid layout
* Footer section

**Concepts Used:**

* Flexbox for layout alignment
* Grid for feature sections
* Responsive design with media queries

---

### 2. Product Page

**Features:**

* Responsive product card grid
* Hover effects on cards
* Clean UI spacing and alignment

**Concepts Used:**

* CSS Grid (`auto-fit`, `minmax`)
* Transitions and hover effects
* Box model and spacing control

---

### 3. Blog Layout

**Features:**

* Sidebar + main content layout
* Article listing grid
* Responsive adaptation for smaller screens

**Concepts Used:**

* Grid layout for page structure
* Flexbox for internal alignment
* Responsive design strategies

---

### 4. Dashboard Layout

**Features:**

* Sidebar navigation
* Nested grid content area
* Structured layout for data representation

**Concepts Used:**

* CSS Grid (nested grids)
* Layout composition
* Component structuring

---

### 5. Portfolio Page

**Features:**

* Navbar and hero section
* Projects showcase grid
* About section
* Footer

**Concepts Used:**

* Combined use of Flexbox and Grid
* Reusable UI components
* Responsive sections

---

## Key Skills Demonstrated

* Strong understanding of **CSS layout systems** (Flexbox & Grid)
* Ability to build **responsive multi-section layouts**
* Use of **semantic HTML structure**
* Implementation of **hover effects and UI interactions**
* Clean spacing using the **box model**
* Basic **component-based design thinking**

---

## Advanced CSS Concepts Applied

* CSS Positioning (relative, absolute, fixed, sticky)
* Stacking context and `z-index`
* Pseudo-classes and pseudo-elements
* Advanced selectors
* CSS variables (custom properties)
* Animations and transitions
* Responsive techniques (`clamp`, fluid layouts)
